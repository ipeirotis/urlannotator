Beat the Machine

Beat the Machine starts a new tagasauris task that will gather urls that may be misjudged by the classifier. If an url is submittable (we don't want urls that category both submiteer and classifier agree on) we send it to actual user validation. Based on the difference on the classifier reslult and user result a bonus will be paid out to the worker that submitted that url. (less)
